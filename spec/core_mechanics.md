# ゲーム仕様書：元素ローグライクカードゲーム（仮称）

## 1. ゲーム概要
- **ジャンル**: ダンジョン探索型 2Dローグライクカードゲーム
- **プラットフォーム/操作方法**:
  - PC（マウス操作）
  - スマートフォン（タッチ操作）
- **ゲームの目的**:
  - 元素カードを駆使して、ダンジョンの最後のマスをクリアすること。

---

## 2. ダンジョン探索システム
- **マップ進行**:
  - プレイヤーは提示されたマスを選択し、奥へと進んでいく。
- **マスの種類**:
  1. **敵との戦闘**: 元素カードを使ったバトルが発生する。
  2. **イベント**: 様々な出来事が発生する。
  3. **ショップ**: 買い物が可能。

---

## 3. バトルシステム（コアメカニクス）
- **カードの構成**:
  - すべてのカードが元素記号（例：`H` 水素、`O` 酸素など）で構成されている。
- **ターンフロー**:
  1. プレイヤーのターン開始時に、手札を6枚引く。
  2. 手札の元素カードを組み合わせて、物質を生成する（例：`H` + `H` + `O` ＝ `H2O` 水）。
  3. 生成された物質に設定された固有の効果が発動し、それによってバトルが進行する。
