# ゲーム仕様書：化合ローグライク LAB

## 1. ゲーム概要
- **ジャンル**: ダンジョン探索型 2Dローグライクカードゲーム
- **プラットフォーム/操作方法**:
  - PC（マウス操作）
  - スマートフォン（タッチ操作）
- **ゲームの目的**:
  - 元素カードを駆使して、ダンジョンの最後のマスをクリアすること。

## 3. ダンジョン探索システム
- **全体構成（エリア進行）**:
  - ゲームは**全3エリア（階層）構成**となっています。
  - 各エリアの最後のマスには、エリアを統括する強力な**ボス**が待ち構えています。ボスを撃破することで次のエリアへ進み、最終エリア（エリア3）のボスを撃破すればゲームクリアとなります。
- **マップ進行**:
  - 分岐があり、一度進むと戻ることができないルート選択型マップ（トーナメント表や樹形図のような構造）。
- **マスの種類**:
  1. **敵との戦闘**: 元素カードを使ったバトルが発生する。
  2. **イベント**: 様々な出来事が発生し、HPの増減やゴールド、アイテム、カードの獲得などの効果が得られる。
  3. **ショップ**: 獲得したゴールドを消費して、新しいカードや実験器具を購入できる。

## 4. ゲーム内リソース
- **ゴールド（ゲーム内通貨）**:
  - **入手方法**: 戦闘の勝利報酬、イベントでの獲得など。
  - **用途**: ショップでの買い物（カード・実験器具の購入）、イベントでの支払いなど。
