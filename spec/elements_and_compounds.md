# 元素カードと合成レシピ（図鑑）

## 1. 元素カード一覧

ゲームに登場する元素カードは、ゲームの進行度（初期所持 / 追加獲得）に応じて分類されます。

### 初期段階で登場（初期デッキ・初期報酬）
- **H（水素）**
- **O（酸素）**
- **C（炭素）**
- **N（窒素）**

### 追加元素として入手可能（戦闘報酬、ショップ、イベントなど）
- 初期デッキには含まれませんが、ゲーム開始直後（最初のマップ）から全てのカードプールに常時追加されています。
- 戦闘後のカード獲得報酬、ショップでの販売、イベントでの獲得など、様々なタイミングでランダムに出現し、入手することができます。
- 対象元素：
  - **Cl（塩素）**
  - **S（硫黄）**
  - **Cu（銅）**
  - **Fe（鉄）**
  - **Na（ナトリウム）**
  - **K（カリウム）**
  - **Ca（カルシウム）**
  - **F（フッ素）**

## 2. 合成図鑑システム

### 概要
- プレイヤーがゲーム中いつでも合成レシピを確認できるシステムです。
- 画面上に「図鑑」ボタンが常時設置されており、バトル中やマップ移動中など、いついかなるときでもすべての合成レシピ（化合物とその効果）にアクセス可能です。

### 開示状態
- 合成レシピは最初から**すべて開示（アンロック）**されています。
- プレイヤーは未知の組み合わせを模索する必要はなく、図鑑を参照しながら戦略的に物質の合成を行うことができます。

### 合成化学式に関する特記事項
- 初期の手札上限は6枚ですが、**実験器具の効果などによって手札上限が増加する要素**が想定されています。
- これにより、手札上限が最大8枚以上となった際に、最大8枚のカード消費を必要とする「酢酸（$CH_3COOH$）」などの高コストかつ強力な化合物も合成可能になります。

### 3. 合成処理ライフサイクルと『効果処理ゾーン』の定義
- **合成素材カードの一時退避挙動**:
  - 手札から元素を合成した瞬間、その合成に使用した素材カードは手札から取り除かれ、プレイヤーからは見えない**「効果処理ゾーン（待位・一時退避領域）」**に一時的に移動します。
  - 化合物の効果（ダメージ、ドロー、シールド、墓地回収モーダル等）がすべて実行中である間、素材カードは**墓地にも手札にも属しません**。
  - そのため、効果処理の途中に発生する「墓地回収（サルベージ）」の対象候補に、いま合成で使用したばかりの素材カードが含まれる（自己回収できてしまう）現象を防ぐことができます。
  - 複数の合成や効果が待機・重なるケースを想定し、効果処理ゾーンのカードは、それぞれどの合成（化合物ID/インスタンス）に起因するかを識別できるよう識別グループ分けして管理されます。
- **効果処理完了後のクリーンアップ処理**:
  - **通常の化合物**: すべての効果が解決された（墓地回収の選択完了、あるいはキャンセルされた）後、効果処理ゾーンに残っている素材カードが正式に**墓地（Grave）**に送られます。
  - **特殊な化合物（例：亜硝酸）**: すべての効果解決後、効果処理ゾーンの素材カードは墓地に行く代わりに**山札（Deck）**に戻され、山札がシャッフルされます。

## 4. 合成化合物（レシピ）一覧と効果
現時点で定義されている化合物の一覧（全51種）です。一部の化合物については、テストプレイ用および初期実装用として、具体的な効果が以下のように決定されています。

### H, O, C, N のみで構成される化合物（初期元素エリア）
1. **水**: $H_2O$ (H: 2, O: 1)
   - **効果**:
     - 敵1体を選択し、対象に**3ダメージ**を与える。
     - 自身に**3のシールド**を付与する。
     - **カードを1枚引く**。
2. **アンモニア**: $NH_3$ (N: 1, H: 3)
   - **効果**:
     - 敵1体を選択し、対象に**4ダメージ**を与える。
     - このターンに「水（$H_2O$）」をすでに合成していた場合、同じ対象に「デバフ：毒」（カウント3）を付与する。
3. **一酸化炭素**: $CO$ (C: 1, O: 1)
   - **効果**:
     - 敵1体を選択し、対象に「デバフ：毒」（カウント1）を付与する。
     - **カードを1枚引く**。
4. **二酸化炭素**: $CO_2$ (C: 1, O: 2)
   - **効果**:
     - 敵1体を選択し、対象に**4ダメージ**を与える。
     - 対象に「デバフ：毒」（カウント1）を付与する。
5. **一酸化窒素**: $NO$ (N: 1, O: 1)
   - **効果**:
     - 自分に**2のシールド**を付与する。
     - 墓地から好きなカードを1枚選び、そのカードを墓地から取り除き手札に加える。
6. **二酸化窒素**: $NO_2$ (N: 1, O: 2)
   - **効果**:
     - 自分に**5のシールド**を付与する。
     - 墓地から好きなカードを1枚選び、そのカードを墓地から取り除き手札に加える。
7. **過酸化水素**: $H_2O_2$ (H: 2, O: 2)
   - **効果**:
     - 敵1体を選択し、対象に**2ダメージ**を与える。
     - 自身に**3のシールド**を付与する。
     - **墓地の水素（H）をすべて山札に戻す（シャッフルする）**。
8. **炭酸**: $H_2CO_3$ (H: 2, C: 1, O: 3)
   - **効果**:
     - 敵1体を選択し、対象に**5ダメージ**を与える。
     - **カードを3枚引き**、その後に**現在（引いた後）の手札の枚数と同じ数だけの「デバフ：毒」を敵1体に付与**する（例：手札が7枚なら毒カウント7を付与）。
9. **シュウ酸**: $H_2C_2O_4$ (H: 2, C: 2, O: 4) (効果未定)
10. **酢酸**: $CH_3COOH$ (C: 2, H: 4, O: 2) ※要・手札上限拡張 (効果未定)
11. **硝酸**: $HNO_3$ (H: 1, N: 1, O: 3)
    - **効果**:
      - 敵1体を選択し、対象に**4ダメージ**を与える。
      - 墓地から好きなカードを**2枚まで選び**、墓地から取り除き手札に加える（墓地の枚数が不足している場合は1枚のみ回収、墓地が空の場合は回収処理を行わない）。
12. **亜硝酸**: $HNO_2$ (H: 1, N: 1, O: 2)
    - **効果**:
      - 敵1体を選択し、対象に**3ダメージ**を与える。
      - 墓地から好きなカードを**1枚選び**、墓地から取り除き手札に加える。
      - **この合成に使用した素材元素カードは、墓地に行く代わりに山札に戻し、山札をシャッフルする**。
