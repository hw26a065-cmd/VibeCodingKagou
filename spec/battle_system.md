# バトルシステム

## 1. プレイヤーと敵の基本ステータス

### プレイヤー
- **ステータス**: HP（体力）のみを持ちます。
- **敗北条件（ゲームオーバー）**: HPが0になる。
- **ゲームオーバー時のルール**: **完全リセット型（パーマデス）**です。進行度、獲得したカードや実験器具、ゴールドはすべて失われ、次回プレイは最初からのスタートとなります。

### 敵（エネミー）
- **ステータス**: HP、およびエネミーの種類に応じた「特性（独自のパッシブ能力や耐性）」を持ちます。
- **敵の特性システム**:
  - 特性はゲームプレイに強く影響します。
  - プレイヤーは、戦闘画面で**敵にカーソルを合わせる（ホバーする、またはタップする）ことで、その敵の特性詳細をダイアログやポップアップで確認**できます。
  - **特性の例**:
    - スライム: 「H（水素）カードが含まれる化合物のダメージを減少させる」
- **勝利条件**: 敵のHPを0にする。

## 2. デッキシステムと山札・墓地ルール

### 初期デッキ構成
ゲーム開始時のデッキは**合計20枚**の元素カードで構成されます。
- **H（水素）**: 6枚
- **O（酸素）**: 6枚
- **C（炭素）**: 4枚
- **N（窒素）**: 4枚

### ドローとリシャッフルの詳細挙動
ターンの開始時など、山札からカードを引く（ドロー）際は、以下のプロセスを厳密に処理します。
1. **1枚ずつのドロー処理**:
   - カードは山札から1枚ずつ引かれます。
2. **山札が空になった場合のリシャッフル**:
   - ドローの途中で山札が0枚になった瞬間、即座に**墓地のカードをすべてシャッフルして新しい山札へと移動**させます。
3. **ドローの継続**:
   - 山札を再構築した後、引くべきだった残りの枚数分、再び1枚ずつ引く行為を完了するまで繰り返します。

## 3. ターンフローと手札ルール

### ターン進行
1. **プレイヤーのターン開始**:
   - 山札からカードを**6枚**、上記「ドロー処理」に基づいて引いて手札とします。
2. **プレイヤーのアクションフェーズ**:
   - 手札の元素カードを組み合わせて物質を合成します（回数制限なし）。
3. **プレイヤーのターン終了**:
   - 合成に使用しなかった余った手札は**すべて墓地へ送られます**（次ターンへの持ち越し不可）。
4. **敵のターン**:
   - 敵があらかじめ予告していた行動を実行します。

### 合成（化合物作成）ルール
- 1ターン中の合成回数に**制限はありません**。手札のカードが足りる限り、何度でも合成を試みることができます。
- 合成に使用した元素カードは、効果発動後にすべて**墓地に置かれます**。
- 例: 手札に「H」「H」「O」がある場合、これらを組み合わせて「H2O（水）」を生成。生成された水に設定された固有効果が発動し、使用した「H」「H」「O」の3枚は墓地へ送られます。

## 4. 敵の行動パターンと予告システム

### 行動予告システム（インテントシステム）
- 敵が次のターンにどのような行動をとるかは、プレイヤーのターン中に**画面上にアイコンやテキスト等で常時予告**されます。
- これにより、プレイヤーは敵の次の行動（大ダメージ、シールド付与など）を事前に把握した上で、手札の元素をどう合成するか戦略を立てることができます。

### 敵の共通行動
敵は予告に基づき、主に以下の行動パターンを組み合わせて実行します。
- **プレイヤーへのダメージ付与**: プレイヤーのHPを減少させる。
- **一時的なシールドの付与**: そのターン限りの防御力を自身に付与し、プレイヤーの攻撃を防ぐ（ターン終了時にシールド値は消滅する）。
- **特殊行動**: 特性に関連した特殊なデバフや強化など。
